Set a configuration property in Odigos

```
odigos config set <property> <value> [flags]
```

## Configurable Properties

The following properties can be set using the `odigos config set` command:

### Boolean Properties

- `telemetry-enabled`: Enables or disables telemetry.
- `openshift-enabled`: Enables OpenShift-specific settings.
- `psp`: Enables or disables Pod Security Policies.
- `skip-webhook-issuer-creation`: Skips the creation of webhook issuers.
- `allow-concurrent-agents`: Allows running multiple agents concurrently.

### String Properties

- `image-prefix`: Specifies a prefix for all container images.
- `ui-mode`: Defines the UI mode, "normal" or "readonly".

### List Properties

- `ignored-namespaces`: Specifies a list of namespaces to be ignored.
- `ignored-containers`: Specifies a list of containers to be ignored.
